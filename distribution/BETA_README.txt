This beta build is proprietary, for approved testers only, and is intentionally
unsigned and not notarized. macOS may block the first launch until you approve
it.

Recommended tester steps:
1. Move "Custom Stem Injector.app" to Applications if desired.
2. Double-click the app once.
3. If macOS blocks it, open:
   System Settings > Privacy & Security
4. Scroll down and click "Open Anyway" for Custom Stem Injector.
5. Confirm the second prompt.

Alternative helper:
- Double-click "Open Custom Stem Injector.command" in this folder.
- It removes the quarantine attribute from the adjacent app and then opens it.

Important notes:
- This build is for Apple Silicon Macs only.
- Intel Macs are not supported by this beta package.
- Because the app is unsigned, Gatekeeper approval is expected on first run.
- Use the app only for private evaluation and tester feedback.
- Do not redistribute this build without permission.
